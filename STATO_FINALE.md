# Viral Space Defender Documentation

## Overview

Viral Space Defender is a new pre-release game that challenges players to defend the spaceship from various viral attacks. This game combines strategy and quick thinking to survive against waves of aggressive viruses.

## Features
- **Engaging Gameplay**: Players navigate through different levels, each with increasing difficulty.
- **Variety of Enemies**: Different types of viruses with unique abilities.
- **Power-ups**: Collect power-ups to gain temporary advantages during gameplay.

## Installation
To get started with Viral Space Defender, clone the repository and install the necessary dependencies:

```bash
git clone https://github.com/Marcone1983/Viral-space-Defender.git
cd Viral-space-Defender
# Install dependencies
```

## How to Play
1. Launch the game using `npm start`.
2. Use the keyboard to control the spaceship.
3. Aim and shoot at incoming viruses to defend your spaceship.
